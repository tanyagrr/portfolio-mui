# Portfolio Website

A personal portfolio website built with React to showcase my projects, skills, and experience in a clean, professional format. The site focuses on usability, responsive design, and clear presentation of information.

## Features

- Project showcase with descriptions and external links
- Responsive layout optimized for desktop and mobile devices
- Client-side navigation using React Router
- Live deployment for easy sharing

## Technologies Used

- React
- JavaScript (ES6+)
- Material UI
- React Router
- Vite
- Deployed with Vercel

## Skills Practiced

- Building reusable React components
- Styling and layout using Material UI
- Client-side routing and navigation
- Responsive design principles
- Project deployment and environment setup

## What I Learned

- How to structure and organize a React application for scalability
- Improved my ability to use Material UI components and theming effectively
- Gained experience with routing to create a smooth, multi-page user experience
- Learned how to build, deploy, and maintain a production-ready React project
